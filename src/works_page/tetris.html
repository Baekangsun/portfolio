Game Site / JavaScript 2D테트리스 게임

2D고전게임 '테트리스' 게임

종강 후 부족한 부분을 공부 중 유튜브를 보면서 Vanilla JavaScript로 테트리스 게임을 하드코딩하여 만들었습니다.
createElement를 사용해 그리드를 그렸으며, 전개 연산자를 사용해 movingitem에
위치를 알 수 있었습니다.
setInterval를 사용해 콘텐츠에 속도를 제어하고, keydown 이벤트를 switch 문에 사용하여 방향 키와 스페이스바 키를 제어할 수 있었습니다.
createElement, 전개 연산자를 경험할 수 있었던 프로젝트였습니다.
