Game Site / JavaScript 2D테트리스 게임

2D고전게임 '테트리스' 게임

유튜브를 보면서 Vanilla JavaScript로 갤러그 게임을 하드코딩하여 만들었습니다. ul 자식 li를 createElement를 사용해 만들어 그리드를 만들었고 전개 연산자를 사용해 movingitem에 위치를 알 수 있었습니다. setInterval을 통해 콘텐츠에 속도를 제어하고 switch 문을 사용하여 keydown 이벤트를 사용하여 방향 키와 스페이스바 키를 제어할 수 있었습니다. 다양한 JavaScript 로직을 접해 보았으며 다른 프로젝트에 응용하여 사용할 수 있습니다.
