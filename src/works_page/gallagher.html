Game Site / JavaScript 2D갤러그 게임

2D고전게임 '갤러그' 게임

종강 후 부족한 부분을 공부 중 유튜브를 보면서 Vanilla JavaScript로 갤러그 게임을 하드코딩하여 만들었습니다.
이미지 데이터를 new 키워드를 사용하여 데이터를 담은 후 keydown 이벤트를 if 문에 사용하여 정해진 키보드를 눌렀을 때 이미지를 그렸습니다.
총알과 적군은 빈 배열에 담아 개수를 늘린 후 적군은 캔버스 상단에서 Math의 random 메서드를 사용해 랜덤 한 위치에서 그렸고, 총알은 우주선의 좌표값에서 그렸습니다.
new 키워드 사용과 Math의 메서드 사용을 경험할 수 있었던 프로젝트였습니다.
